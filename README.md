# Number-Guessing-Game

This is a number guessing game that is written in Python where the user attempts to guess a number between 1 and 100 in a certain amount of guesses.

## Requirements

- Python 3.x

## Usage
```bash
python number_guessing_game.py
```

Select a difficulty when prompted by the code:
- Easy Mode(10 Guesses)
- Medium Mode(5 Guesses)
- Hard Mode(3 Guesses)
